# 博主接商单

不是教你等品牌方找你，是帮你判断现在该做什么，才能让合作找上门。

很多博主以为接商单要等粉丝过万。这是错的。品牌方找博主的核心逻辑是：这个博主的内容能触达我的目标用户吗，性价比够高吗？粉丝数只是参考之一，垂直赛道、互动率、内容与产品匹配度有时候更重要。

这个 Skill 帮你判断账号当前阶段、主动出击找合作、筛选邀约、谈价和写报价单。

## 这个 Skill 适合谁

- 想开始接商单但不知道从哪里下手的博主
- 粉丝量不大但想找垂直品牌合作的创作者
- 收到合作邀约不知道该不该接、怎么回
- 不知道自己该报多少价

## 它能做什么

- 从四个维度判断账号现在的接单准备度
- 给出三个阶段（0-500/500-3000/3000+粉）的不同策略
- 用真实数据框架帮你估合理报价区间
- 给出筛选合作邀约的五条标准
- 提供主动发给品牌的合作邀约话术
- 给出收到邀约后的标准回复模板

## 典型使用场景

- "怎么接商单，从哪里开始"
- "我现在有多少粉丝可以接单了"
- "收到品牌合作邀约，该怎么回"
- "报价多少合适"
- "怎么主动找品牌合作"

## 推荐输入格式

```text
1. 账号方向（如：AI工具测评）
2. 目前粉丝数
3. 近期内容的大概数据（播放/收藏/互动）
4. 遇到的具体问题（接单/报价/邀约回复）
```

## 输出长什么样

默认输出：

1. 接单准备度评估（四维度 + 当前阶段判断）
2. 对应阶段的具体行动建议
3. 报价区间估算（基于真实数据）
4. 邀约筛选标准
5. 可直接用的回复话术或主动邀约模板
