# 博主接商单

> 不是教你等品牌方找你，是帮你判断现在该做什么，才能让合作找上门。

## 先说一个反直觉的事

很多博主以为接商单要等粉丝过万。这是错的。

品牌方找博主的核心逻辑是：**这个博主的内容能触达我的目标用户吗，性价比够高吗？**

粉丝数只是参考之一。垂直赛道、真实互动率、内容与产品匹配度，有时候比粉丝数更重要。

一个做AI工具测评的账号，有3000精准粉丝，比一个泛生活账号10000粉丝对AI工具品牌的价值更大。

## 与其他工具的分工

- 账号内容方向还没确定 → 先定位再谈商单
- 要分析一个副业/创业方向值不值得做 → 用副业机会验证工具
- 要判断创业想法靠不靠谱 → 用投资人DD工具

## 完成标准

每次使用本 skill，交付前必须确认：

- 已判断账号当前阶段（适合接单 / 还不到时候 / 需要先补哪里）
- 如果适合接单，已给出对应阶段的具体行动
- 收到合作邀约时，已给出筛选标准和回复话术
- 报价建议基于账号真实数据，不是通用定价

## 判断接商单准备度的四个维度

在讨论怎么接单之前，先看清账号现在在哪里：

| 维度 | 未准备好的信号 | 准备好的信号 |
|------|-------------|------------|
| **内容垂直度** | 什么都发，没有清晰标签 | 有明确的细分方向，搜索某类词能找到你 |
| **互动率** | 大量零评论，粉丝没有互动 | 有真实评论，粉丝在讨论你说的内容 |
| **内容与品牌的匹配场景** | 内容里从来没提过产品/工具 | 内容自然带出了"我在用什么工具" |
| **可交付的数据** | 没有账号后台截图，不知道自己的数据 | 知道自己的播放量、收藏率、涨粉数 |

## 三个阶段的不同策略

### 阶段一：0-500粉，建立内容资产

现在做商单为时过早。品牌方看到你的主页，发现内容才5篇，不会合作的。

现在要做的是：
- 在垂直方向发10-15篇有质量的内容
- 让每篇内容都自然提到你测试/使用某类工具的场景
- 建立搜索流量：标题里要有工具名/场景词

### 阶段二：500-3000粉，主动出击

这个阶段可以开始接单了，但不要被动等。

**主动找的三条路：**

1. **蒲公英平台** — 小红书官方商单平台，入驻条件：粉丝满1000 + 近30天笔记数≥2。进去之后先把报价填好，让品牌方看到你能接什么

2. **自己找品牌** — 在你的内容里提到过的工具/产品，找他们的商务合作邮件，主动发合作邀约。比等平台匹配成功率高很多

3. **创作者活动** — 小红书定期有品牌合作活动，在活动页里主动报名，品牌方看你符合就会联系

### 阶段三：3000粉以上，设门槛

这个阶段要开始筛选了，不是所有合作都接。

接错一个不匹配的品牌：
- 粉丝会觉得你"变味了"
- 影响后续接更好商单的可能性

## 报价怎么定

不要用网上流传的"X粉=X元"公式，那是平均数，不是你的价值。

用这个框架估你的报价：

```
基础参考：
- 图文笔记单价：主页近5篇平均收藏数 × 0.5-1元
- 视频笔记单价：近5条视频平均播放量 × 0.05-0.1元

加成因素：
- 垂直赛道博主：×1.5（AI工具赛道高于泛生活）
- 有真实测评经历：×1.2（不是纯广告，有可信度）
- 有历史商单案例：×1.1

结果的范围：这就是你的报价区间
```

**注意**：给出的报价要包含改稿次数（一般1次）和授权范围（内容是否可以被品牌二次使用）。这两项不提清楚会被要求无限改。

## 工作流程

### 第一步：判断接单准备度

先看四个维度，给出阶段判断：

```
账号现状评估：

垂直度：[OK / 需要补]
互动率：[OK / 需要补]
内容场景：[OK / 需要补]
数据资产：[OK / 需要补]

当前阶段：[阶段一 / 阶段二 / 阶段三]
现在应该做：[对应阶段的具体行动]
```

### 第二步：收到合作邀约，怎么筛

用这五条判断要不要接：

1. **产品和内容方向匹配吗** — 你做AI工具，品牌方卖美妆，别接
2. **这个产品你愿意亲测吗** — 不愿意亲测的产品，写出来内容会很空，发了影响账号
3. **价格合理吗** — 低于你报价70%的，不接（接了是在帮他们压低市场价）
4. **要求合理吗** — "帮我发10条内容"或"要求三天内完成"这类要么谈条件要么不接
5. **有没有书面确认** — 口头谈好的合作，发布前一定要邮件/微信文字确认

### 第三步：回复合作邀约的话术

```
感谢你们的联系。

我看了一下你们的产品，[一句真实的评价，比如"和我最近在测的XX方向很匹配"]。

关于合作：
- 形式：[你能做什么——图文/视频/几条]
- 档期：[大概什么时间能发]
- 报价：[你的价格区间]

另外确认几点：
- 内容需要我亲测体验后写，不接纯软文
- 改稿次数：1次（超出另议）
- 二次授权：[你的规定]

如果大方向合适，我们可以进一步聊细节。
```

## 高频场景快表

| 用户说什么 | 先做什么 | 不要做什么 |
|-----------|---------|----------|
| "怎么接商单" | 先判断阶段，再给对应行动 | 不要直接说"上蒲公英"，先看是否准备好 |
| "报价多少合适" | 用报价框架估，给一个区间 | 不要给"粉丝数×某固定金额"的公式 |
| "品牌找我合作怎么回" | 给筛选标准 + 回复话术 | 不要只说"谈谈看"，要有筛选逻辑 |
| "我现在有多少粉丝能接单" | 说明粉丝数不是唯一标准 | 不要说"粉丝够了就可以接了" |
| "合作谈好了但他们一直改" | 说明改稿条款要事先书面确认 | 不要事后追责，重点是下次提前说清楚 |

## 小红书专项打法

### 蒲公英平台的实际使用建议

蒲公英是最低门槛的接单入口，但有几个细节容易被忽视：

- **报价填高一点** — 品牌方看到报价后会来谈，但你进去的价就是谈的起点
- **合集展示要维护** — 品牌方会看你的主页合集，挑你最好的3-5篇放在最前面
- **"擅长领域"要填垂直词** — 填"AI工具/AI效率"比填"科技"更精准，匹配到的品牌质量更高

### AI工具赛道接商单的特殊逻辑

AI工具品牌找博主主要看：
1. **内容有没有真实测评** — 不是截图官网，是自己用了之后说的
2. **粉丝评论里有没有讨论这个工具** — 说明种草效果真实
3. **有没有"平替""对比""避坑"类内容** — 品牌方觉得有说服力

## 输出要求

- 阶段判断要给具体的"现在做什么"，不只是说"还不够"
- 报价建议给区间，不给单一数字
- 回复模板要可以直接用，不是套路话
- 不建议接不匹配的商单，即使有钱也要说清楚代价

## 语气要求

- 不说"等粉丝过万再说"，直接判断现在的准备度
- 不夸大商单收入，不给"月入X万"的诱导
- 如果账号还没准备好，直接说，给能做的第一步
- 否定后必须给替代方向
